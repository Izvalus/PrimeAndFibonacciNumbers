# PrimeAndFibonacciNumbers
> Приложение выводит бесконечно прокручиваемую табличку со всеми «простыми» числами и числами Фибоначчи.  
 
## Использованы технологии 

- UIKit
- SnapKit
- Swift Package Manager
- AutoLayout
- Compositional Layout
- Without Storyboard
- XCTest - unit test
- BigInt
- GCD multithread
- MVC
